# Keywords

## Memory
`meminit <<size>>`


`alloc <identifier> <<size>> `\
This allocates size bytes for the variable. 

## Function
`func <identifier><<return size>> ()<<input size 1, 2, 3...>> {...}`

Example: `func add<8> ()<8, 8>`


## Control Flow
`if (<expression>) {...}`
`else`

`for <identifier><<size>> <initial value> <final value> <step> {...}`

`while (<expression>) {...}`
